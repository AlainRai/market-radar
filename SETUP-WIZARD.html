STEP 4 OF 5
Run the bot for the first time
This is where everything comes to life. Open a terminal on your computer and run these commands.
💻
On your computer — open a Terminal (or Command Prompt)
1
Create the project folder and get the files
If you haven't already — create a folder called market-radar and put all the bot files inside a bots/ subfolder. Make sure .env is in that bots/ folder.
2
Install Python dependencies
cd market-radar/bots pip install -r requirements.txt
3
Run the bot — first cycle
python main_bot.py
You'll see live output like this:

❓
If you see errors
🔴
ModuleNotFoundError
Run: pip install -r requirements.txt (again, in the bots/ folder)
🔴
AuthenticationError from Anthropic
Your ANTHROPIC_API_KEY in .env is wrong — check for spaces/extra characters
🔴
Supabase connection failed
Double-check SUPABASE_URL ends in .supabase.co and SUPABASE_SERVICE_KEY is the service_role key, not anon
🟡
No data for some stocks
Normal — yfinance occasionally fails. The bot handles it gracefully and continues
